# Design System & Aesthetic Rules (Magic Oven)

Este documento define os padrões visuais, tokens e princípios de UI/UX para o ecossistema Magic Oven (Site Público e CMS).

## 1. Estética e Filosofia
O projeto segue uma estética de **Brutalismo Moderno** misturada com elementos **Cyberpunk** e **Experimental**.
- **Contraste Alto:** Fundo preto profundo com acentos em verde vibrante.
- **Textura Digital:** Uso de efeitos de dithering (Bayer) e glitch para evocar instabilidade digital controlada.
- **Tipografia Brutalista:** Uso de fontes sem-serifa geométricas e fontes monoespaçadas para dados técnicos.

## 2. Design Tokens (Cores)
As cores devem ser referenciadas preferencialmente via variáveis CSS (`--color-*`).

- **Background:** `#080808` (`--color-background`)
- **Surface:** `#0d0d0d` (`--color-surface`)
- **Surface Elevated:** `#121212` (`--color-surface-elevated`) - *Usado em cards e modais.*
- **Accent (Cyber Green):** `#27FF2B` (`--color-accent-primary`) - *Usado para CTAs, estados de sucesso e focos.*
- **Error/Danger:** `#ff3b30` - *Usado para exclusões e estados críticos.*
- **Text Primary:** `#ffffff` (`--color-text-primary`)
- **Text Secondary:** `#a0a0a0` (`--color-text-secondary`)

## 3. Tipografia
- **Títulos e UI Principal:** `Space Grotesk` (Sem-serifa, moderna).
- **Dados e Código:** `JetBrains Mono` (Monoespaçada).

## 4. Componentes de UI (Padrões CMS)

### Item Cards (Listagens)
Todas as listagens no CMS (Blog, Projetos, Mensagens) devem seguir o layout de grid de 3 colunas:
1. **Seleção:** Checkbox para ações em lote.
2. **Informação:** Conteúdo principal (Título, Metadados, Status).
3. **Ações:** Botões de ação rápida agrupados à direita.

### Indicadores de Status
- Usar o `.status-indicator` (círculo pequeno) próximo ao título.
- **Verde (`--color-accent-primary`):** Publicado, Lida, Online.
- **Amarelo (`#ffcc00`):** Rascunho, Pendente, Offline.

### Botões de Ação
- Priorizar `btn-icon` com SVGs minimalistas dentro de listagens.
- Usar `btn-primary` para ações afirmativas de criação (Nova publicação).
- Usar `btn-outline` para ações secundárias ou de cancelamento.

## 5. Efeitos Visuais
- **Glow:** Aplicar `box-shadow` ou `filter: drop-shadow` suave em verde apenas em elementos de extrema importância.
- **Transições:** Todas as interações de hover devem usar a variável `--transition-base`.
